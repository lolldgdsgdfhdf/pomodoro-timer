# AGENTS Instructions

- 這是番茄鐘計時器專案，包含 CLI (`pomodoro_timer.py`) 和 GUI (`pomodoro_gui.py`) 兩個版本
- 共用統計模組：`pomodoro_stats.py`（`get_stats()` 單例）
- 運行測試：`python test_counter.py` 和 `python test_pomodoro_stats.py`
- Python 3，無外部依賴（GUI 用內建 tkinter）
- Windows 終端需 `PYTHONIOENCODING=utf-8` 來正確顯示 emoji
- 改完程式碼後務必跑兩個測試檔確認沒有破壞現有功能
